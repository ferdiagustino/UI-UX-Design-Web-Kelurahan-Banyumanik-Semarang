# UI-UX-Design-Web-Kelurahan-Banyumanik-Semarang
UI/UX design of Kelurahan Banyumanik website created using Figma. This project focuses on designing a user-friendly interface for public information services, including homepage, navigation, and service features.

# 🏢 Desain UI/UX Website Kelurahan Banyumanik

## 📌 Gambaran Umum
Proyek ini merupakan desain UI/UX website Kelurahan Banyumanik yang dibuat menggunakan Figma. Desain ini bertujuan untuk menghadirkan tampilan website yang modern, informatif, dan mudah digunakan oleh masyarakat dalam mengakses layanan serta informasi publik.

## 🎯 Tujuan
- Meningkatkan aksesibilitas informasi publik
- Membuat tampilan website instansi yang modern dan rapi
- Memberikan pengalaman pengguna (user experience) yang baik

## 🧩 Fitur Desain
- Halaman Beranda (Homepage) dengan banner informasi
- Navigasi menu (Beranda, Profil, Berita, Layanan, Informasi Publik, dll)
- Desain fitur pencarian (Search Bar)
- Tampilan layanan publik
- Layout yang terstruktur dan user-friendly
- Konsep desain responsif

## 🛠️ Tools & Teknologi
- Figma (UI/UX Design)

## 🎨 Pendekatan Desain
Desain ini dibuat dengan fokus pada:
- Tampilan sederhana dan mudah dipahami
- Navigasi yang jelas dan terstruktur
- Visual yang sesuai dengan instansi pemerintahan
- Kemudahan akses untuk semua kalangan pengguna
